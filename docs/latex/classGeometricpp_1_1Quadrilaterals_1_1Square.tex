\doxysubsection{Descrição detalhada}
Representa um quadrado definido por um lado. 

Esta classe encapsula a lógica para calcular propriedades de um quadrado, que é uma forma geométrica com todos os lados de igual comprimento.

A classe possui os seguintes métodos\+:
\begin{DoxyItemize}
\item size()\+: Retorna o comprimento do lado do quadrado.
\item area()\+: Calcula e retorna a área do quadrado.
\item perimeter()\+: Calcula e retorna o perímetro do quadrado.
\item diagonal()\+: Calcula e retorna o comprimento da diagonal do quadrado.
\end{DoxyItemize}
