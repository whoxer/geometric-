Construtor para a classe Rectangle.

Este construtor inicializa um objeto da classe Rectangle com os valores fornecidos para altura (h) e largura (w). Se os valores de altura e largura forem iguais, uma exceção do tipo std::invalid_argument é lançada, garantindo que o retângulo não seja um quadrado.

Parâmetros
hAltura do retângulo.
wLargura do retângulo.
Excepções
std::invalid_argumentSe os valores de altura e largura forem iguais.
