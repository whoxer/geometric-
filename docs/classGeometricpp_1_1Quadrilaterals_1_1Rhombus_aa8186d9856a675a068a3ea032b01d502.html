double_t Geometricpp::Quadrilaterals::Rhombus::height ( ) const

Calcula a altura do losango.

Esta função calcula e retorna a altura do losango, utilizando a fórmula altura = área / comprimento do lado. A área é calculada com base nas diagonais do losango, e o comprimento do lado é obtido utilizando o Teorema de Pitágoras através da função pitagoras_diagonais().

Retorna
double_t Altura do losango.

Referências Geometricpp::pitagoras_diagonais().
