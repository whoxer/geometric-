Calcula a hipotenusa de um triângulo retângulo usando o Teorema de Pitágoras.Essa função recebe os comprimentos de dois catetos e retorna o comprimento da hipotenusa de um triângulo retângulo. A fórmula utilizada é: , onde e são os catetos e é a hipotenusa.

Parâmetros de template
TTipo numérico (como int, float, double) dos catetos e da hipotenusa.
Parâmetros
aComprimento do primeiro cateto.
bComprimento do segundo cateto.
Retorna
O comprimento da hipotenusa do triângulo retângulo.
Nota
A função utiliza a função sqrt da biblioteca cmath para calcular a raiz quadrada.

Exemplo de uso da função: double hipotenusa = pitagoras(3.0, 4.0); std::cout << "A hipotenusa é: " << hipotenusa << std::endl; // Saída: 5

template <typename length>
typename std::enable_if<std::is_arithmetic<length>::value, length>::type
pitagoras_diagonais(length largest_diagonal, length smallest_diagonal)
{
if (largest_diagonal < smallest_diagonal)
{
throw std::invalid_argument("largest_diagonal parameter cannot be smaller than smallest_diagonal parameter");
}
return sqrt((largest_diagonal / 2) * (largest_diagonal / 2) +
(smallest_diagonal / 2) * (smallest_diagonal / 2));
}
template <typename T>
typename std::enable_if<std::is_arithmetic<T>::value, T>::type
pitagoras(T a, T b)
{
return std::sqrt(a * a + b * b);
}

double_t angle_radians = angle * PI / 180.0;
diagonal_a = side * sqrt(2 * (1 + cos(angle_radians)));
diagonal_b = side * sqrt(2 * (1 - cos(angle_radians)));
}
