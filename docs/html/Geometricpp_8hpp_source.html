39  /* Teorema de Pitágoras aplicada a Diagonais
40  *
41  * Calcula e retorna o comprimento dos lados usando as diagonais
42  * de quatro triângulos retângulos.
43  *
44  * exemplo:
45  *
46  * L² = (D / 2)² + (d / 2)²
47  *
48  *
49  * ...
50  */
51  template <typename length>
52  typename std::enable_if<std::is_arithmetic<length>::value, length>::type
53  pitagoras_diagonais(length largest_diagonal, length smallest_diagonal)
54  {
55  if (largest_diagonal < smallest_diagonal)
56  {
57  throw std::invalid_argument("largest_diagonal parameter cannot be smaller than smallest_diagonal parameter");
58  }
59 
60  return sqrt((largest_diagonal / 2) * (largest_diagonal / 2) +
61  (smallest_diagonal / 2) * (smallest_diagonal / 2));
62  }

75  /*
76  * O namespace ``Quadrilaterals`` é responsável por agrupar as funções que
77  * realizam cálculos relacionados a objetos geométricos planos do tipo quadrilátero.
78  * Quadriláteros são figuras geométricas com quatro lados e ângulos internos que
79  * somam 360 graus, sendo representados por tipos específicos como o "Quadrado"
80  * e o "Retângulo", entre outros.
81  *
82  * Este namespace serve para organizar o código de maneira lógica, separando as
83  * funções de cálculo associadas a quadriláteros em uma única área. Isso facilita
84  * a manutenção do código e evita conflitos de nomes com outras funções ou classes
85  * que não estão relacionadas a quadriláteros.
86  *
87  * Funções dentro deste namespace podem incluir:
88  * - Cálculo de área: diferentes fórmulas para quadrados, retângulos e outros tipos
89  * de quadriláteros.
90  * - Cálculo de perímetro: soma dos comprimentos de todos os lados de um quadrilátero.
91  * - Verificação de propriedades: funções que verificam se um quadrilátero é regular,
92  * ou se os lados opostos são paralelos, por exemplo.
93  *
94  * Ao agrupar essas funcionalidades no namespace "Quadrilaterals", o código se torna
95  * modular e organizado, permitindo fácil expansão e reutilização. Um usuário da
96  * biblioteca pode acessar todas as funções relacionadas a quadriláteros usando o
97  * prefixo "Quadrilaterals::", o que melhora a legibilidade e claridade do código.
98  */
